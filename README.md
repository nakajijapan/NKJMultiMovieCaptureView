## Usage

NKJMultiMovieCaptureView higher requires Xcode 5, targeting either iOS 7.1 and above, or Mac OS 10.9 OS X Mavericks and above.

* AVFoundation.framework
* CoreMedia.framework

## Installation

### CocoaPods

```
pod "NKJMovieComposer"
```
